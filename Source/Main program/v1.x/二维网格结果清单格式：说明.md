**注意：本格式约定与0.x版本的一维结果清单格式不兼容，是互相独立的结果清单格式。**

## 格式

本数据格式有如下字段：

- 受测试的算法名称
- 二维网格的尺寸：元组`(len(A), len(B)`，仅用于读取时方便指示
- 数据值：索引坐标和速度的三元组
- A测试用例集
- B测试用例集



## 样例

```python
result_map = 
{
    "name": "RD",
    "size": (2, 3),  # 该字段仅为可读性写入，目前没有使用
    "value": [(0, 0, 0.0001), (0, 1, 0.0002), ...],
    "A": [15, 16, 17, 18, 19, 20],
    "B": [204, 303, 402, 501, 600, 699],
}
```



## 备注

1. 使用`.json`格式文件存储
2. 后缀可以是`.2drm`（2D Result Map）
